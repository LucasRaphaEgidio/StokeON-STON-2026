## 🚀 Sobre o Projeto

- O Depósito do Zé é um sistema web voltado para o gerenciamento completo e otimizado de uma loja de materiais de construção. Inspirado em uma versão anterior desenvolvida em Java Desktop, o novo sistema foca em acessibilidade, responsividade (uso via mobile e navegadores modernos) e controle eficiente de estoque e vendas.

## 🎯 Objetivo

- Proporcionar uma ferramenta moderna, rápida e eficiente para o controle das operações diárias de um depósito, garantindo a integridade dos dados (como o bloqueio de estoque negativo) e facilitando o trabalho dos funcionários no momento da venda.

## ⚙️ Principais Funcionalidades

- Cadastrar, editar e excluir Clientes

- Cadastrar, editar e excluir Funcionários

- Cadastrar e categorizar Produtos (Alvenaria, Ferragens, Carpintaria, etc.)

- Realizar Novas Vendas (Frente de Caixa/PDV)

- Controle automático e baixa de estoque

- Gerar notas fiscais e relatórios

## 👥 Público-Alvo

- Gerentes, vendedores e estoquistas de lojas de materiais de construção que precisam de um sistema ágil para controlar inventário e registrar vendas de forma simples.
